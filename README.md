# Project Epson QC - Capstone A3

## Cara Menjalankan Project:

### 1. Frontend (React + Vite)
- Masuk ke folder `qc-frontend`
- Ketik `npm install`
- Ketik `npm run dev`

### 2. Backend (FastAPI)
- Masuk ke folder `capstone-a3-backend`
- Buat virtual environment `python -m venv venv`
- Install library `pip install -r requirements.txt`
- Jalankan server `uvicorn main:app --reload`
